一个程序员的软件

工欲善其事，必先利其器。列举一下我平时开发时候用到的软件。

完整软件

  • Licecap
    简单易用的录制gif的软件.

  • Charles
    Http&Https抓包利器.

  • iTerm2
    好用的命令行工具.

  • JSON Accelerator
    可以把json字符串直接转换成 bean 类文件.

  • BetterToucTool
    快捷按键定制神器

软件插件

  • FireShot
    Chrome插件，可以滚动截图网页.

  • 印象笔记-剪藏
    一键收藏网页内容
